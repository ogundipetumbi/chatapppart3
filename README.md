# ChatApp Part 3

## Features Implemented

- JSON message storage using Gson
- Parallel arrays for sent, stored, disregarded messages, hashes and IDs
- Stored Messages menu and submenu
- Search for a message using Message ID
- Search for messages by recipient
- Delete messages using a message hash
- Display the longest message
- Display stored messages report
- JUnit 5 unit testing
